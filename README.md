# How to Run :
- Clone the repo using `git clone git@github.com:Levi477/Black-Pearl.git`
- then `cd Black-Pearl && npm i`
- Go to https://github.com/Levi477/Pirated-Game-Website-Scrapper for downloading scrapper.js file
- put desired scrapper.js into `extensions` folder in the Black-Pearl project folder
- run `npm run dev` to launch the app

## ✨ Features

Black Pearl isn't just a launcher; it's a fully integrated ecosystem designed for performance and aesthetics.

* **⚡ Integrated Aria2 Download Engine:** Built-in multi-threaded downloading with background polling, auto-resume, and stealth session interception.
* **🧩 Dynamic Extension System:** Add your own sources! Black Pearl dynamically loads `.js` extensions from your userdata folder to populate the homepage, categories, and search.
* **🎮 Smart Game Launcher:** Automatically detects when a game launches and closes (even if it uses an intermediate launcher wrapper). Supports custom launch parameters.
* **💨 Lite Mode for Low-End PCs:** Instantly toggle off all heavy blur effects, CSS shadows, and Framer Motion spring animations for maximum CPU/RAM savings.
* **🌐 Steam Metadata Integration:** Automatically fetches high-quality media galleries, descriptions, release dates, and system requirements from the Steam API.

## 🚀 Getting Started

Want to run Black Pearl locally or contribute to the project? 

### Prerequisites
* [Node.js](https://nodejs.org/) (v20 or higher)
* Git

### Installation

- Clone the repo using `git clone git@github.com:Levi477/Black-Pearl.git`
- then `cd Black-Pearl && npm i`
- run `npm run dev` to launch the app
- Go to https://github.com/Levi477/Black-Pearl-Scrappers for raw scrapper.js file url
- put desired scrapper.js raw url into `add extensions`section in the Black-Pearl App
